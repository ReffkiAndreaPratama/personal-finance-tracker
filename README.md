# 💰 Personal Finance Tracker

Personal Finance Tracker is a simple web-based application designed to help users manage their daily finances efficiently.  
This project allows users to record income and expenses, monitor balances, and view financial transaction history in one place.

---

## 📌 Features

- ➕ Add income transactions
- ➖ Add expense transactions
- 📂 Transaction categories
- 📊 Balance and financial summary
- 📜 Transaction history
- 🔍 Search and filter transactions
- 📱 Responsive design
- 💾 Data storage support

---

## 🚀 Installation Guide

### 1. Clone Repository

```bash
git clone https://github.com/yourusername/personal-finance-tracker.git
```

### 2. Open Project Folder

```bash
cd personal-finance-tracker
```

### 3. Run the Project

#### If using HTML/CSS/JavaScript only
Open `index.html` in your browser.

#### If using PHP

```bash
php -S localhost:8000
```

Then open:

```bash
http://localhost:8000
```
